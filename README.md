**clone the Repository**  
```
git clone[https://github.com/namratagite13/Email-Sender-App/tree/main]
cd [EMAIL_SENDER-1]  
```
**Install Dependencies**  
```
npm install
```
### Running Application  
**start server**  
```
node server.js
```
or  
```
npm start
```
**open the apllication in your browser**  
`http://localhost:3000`
